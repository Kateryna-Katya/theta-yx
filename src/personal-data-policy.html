Политика обработки персональных данных

1. Общие положения

Настоящая политика обработки персональных данных (далее — «Политика») определяет порядок и условия обработки персональных данных, предпринимаемые компанией Theta-YX (далее — «Оператор»), и устанавливает меры по обеспечению безопасности этих данных.

1.1. Важнейшей целью Оператора является соблюдение прав и свобод человека при обработке его персональных данных, в том числе защиты прав на неприкосновенность частной жизни.

1.2. Настоящая Политика применяется ко всей информации, которую Оператор может получить о посетителях веб-сайта Theta-YX.

2. Основные понятия

  • Веб-сайт — совокупность материалов и программ, обеспечивающих их доступность в сети интернет по адресу Theta-YX.
  • Пользователь — любой посетитель веб-сайта.
  • Персональные данные — любая информация, относящаяся к Пользователю.

3. Данные, которые мы обрабатываем

Оператор может обрабатывать следующие данные:

  • Фамилия, имя;
  • Адрес электронной почты (Email);
  • Номера телефонов;
  • Обезличенные данные о посетителях (в т.ч. файлы «cookie») с помощью сервисов интернет-статистики.

4. Цели обработки персональных данных

  • Идентификация Пользователя.
  • Установление обратной связи и обработка заявок.
  • Информирование Пользователя посредством отправки электронных писем.

5. Порядок сбора и безопасности данных

  • Оператор принимает все возможные меры для защиты данных от неправомерного доступа.
  • Персональные данные Пользователя никогда не будут переданы третьим лицам, за исключением случаев, связанных с законодательством.

6. Заключительные положения

Пользователь может получить любые разъяснения по вопросам обработки данных, обратившись к Оператору по электронной почте hello@theta-yx.blog.

Актуальная версия Политики всегда доступна на сайте Theta-YX.
